# Create the GitHub Repository

## Recommended Repository Name

`AI-Predictive-Resource-Optimization`

## GitHub Web Method

1. Sign in to GitHub.
2. Select **New repository**.
3. Enter the repository name:
   `AI-Predictive-Resource-Optimization`
4. Set visibility to **Public**.
5. Do not initialize with a README, `.gitignore`, or license.
6. Create the repository.
7. Extract this ZIP package.
8. Upload the extracted files and folders to the new repository.
9. Commit the upload.
10. Confirm that the notebook, README, data folders, docs, images, and reports are visible.
11. Copy the public GitHub URL into the synopsis.

## Git Command Method

```bash
git init
git add .
git commit -m "Initial QM640 capstone synopsis repository"
git branch -M main
git remote add origin https://github.com/YOUR-USERNAME/AI-Predictive-Resource-Optimization.git
git push -u origin main
```

## Important

Do not upload the complete Bitbrains raw dataset. The notebook downloads it from the official public source. The `.gitignore` file is already configured to exclude it.
