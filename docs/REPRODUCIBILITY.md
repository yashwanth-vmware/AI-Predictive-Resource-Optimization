# Reproducibility

The complete workflow is executed from the single Google Colab notebook:

```text
notebooks/01_Capstone_Project_Workflow.ipynb
```

The notebook:

1. mounts Google Drive;
2. creates the workspace;
3. downloads the official Bitbrains dataset;
4. validates the trace files;
5. produces the required dataset evidence;
6. generates figures and reports.

The full dataset is excluded from GitHub. Reviewers can reproduce the work by running the notebook and allowing it to download the official dataset from the documented public source.
